1. What is the difference between the usage of HTML and CSS?

    HTML is used to structure the content on the web page.

    CSS is used to add style to the content of a web page.

  2. What is the newest version of HTML?

    HTML 5

  3. What are the top five most used HTML5 elements?

    Top five most used HTML5 tags:

    1. <header>
      2. This tag was made to replace the <div id = "header">, lots of programmers still use today because of how much easier this was to type and style.
    2. <canvas>
      3. This tag revolutionized web-game developing and is so widely used that it could soon take over Flash. It supports multiple forms of code, one of the most widely used being WebGL.
    3. <nav>
      4. This tag has absolutely wrecked the market and is competing very closely with the first two. This tag happened to replace the <div id = "nav"> tag that used to be widely used and is still often used today.
    4. <section>
      5. This tag is not as widely used as the <div> tag, but still helps people organize articles and other things.
    5. <aside>
      6. This tag is used for small segments that show up on the side of the page (of course, it could be used for anything like a <div>)

  5. What is the difference between semantic and non-semantic elements?

    Semantic elements are elements with meaning. These elements describe the type of content they are supposed to contain.

    Non-semantic elements don’t have any meaning. They don’t tell anything about the content they contain. They can be used with different attributes to mark up semantics common to a group.

  6. Why do we prefer using semantic elements?

    Semantic elements make the HTML more comprehensible by better defining the different sections and layout of web pages. It makes web pages more informative and adaptable, allowing browsers and search engines to better interpret content.
